Django在服务器中常用的操作

  1. 查询端口占用
    查看端口占用 # netstat -anp ｜ grep <port>
    杀死相关进程 # kill -9 <pid>
    systemctl restart memcached
    systemctl status memcached
  2. Crontab
    系统里经常有一些定时任务需要更新和运行，对于django来说我使用django-crontab来配置一些定时运行的检查脚本
    首先通过pip install django-crontab
    然后在INSTALLED_APPS中添加’django_crontab’ # 在app的前面

CRONJOBS可以写成配置文件也可以写在settings里

  • 还有一些在命令行中的操作
    python manage.py crontab add
    python manage.py crontab show
    python manage.py crontab remove
