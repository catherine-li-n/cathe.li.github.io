docker部署vue项目-小记

首先将本地的vue项目打包
npm run build:prod
将打包后的dist文件拷贝到一个新目录下，比如/项目地址/frontend/

在docker中拉取nginx镜像

项目镜像基于nginx基础镜像构建
docker pull nginx

在新目录下新建Dockerfile

vim Dockerfile
dockerfile内容如下

然后在目录下新建nginx.conf

Docker新建镜像

切换到新建目录下，运行指令
docker build -t 镜像名 .
不要忘掉这个‘点’

运行镜像

docker run -d -p 8000:80 镜像名
-d表示后台运行 运行后会返回一个很长的进程号
-p表示端口号

查看运行镜像

docker ps
如果进程正常启动就可以在浏览器里访问了
