Element-ui 使用el-table 并且使某个单元格editable

当我们用table展示数据的时候，对数据的修改一般是以row为单位的
而修改当前row的某一个字端的时候，当前row的key是必要的，如果是link我们可以把id作为参数写进去
但如果在页面上注册change事件，就突然不知道咋处理了。
目前的方案是直接传入一个row对象，这样本行相关数据都可以取到，更新起来也比较方便。
